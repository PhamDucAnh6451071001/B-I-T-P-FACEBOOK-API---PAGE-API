Chính sách quyền riêng tư

Ứng dụng pageapi dùng Facebook Graph API để quản lý Page Facebook và tự động trả lời bình luận theo quy tắc đã cấu hình.

Dữ liệu thu thập

  • Thông tin Page (tên, ID, bài viết, bình luận) qua quyền Page mà admin cấp.
  • Nội dung bình luận/tin nhắn nhận qua webhook để xử lý phản hồi.

Cách sử dụng dữ liệu

Dữ liệu chỉ dùng để vận hành tính năng quản lý Page, phân loại ý định và gửi phản hồi. Không bán hoặc chia sẻ cho bên thứ ba.

Lưu trữ

Sự kiện được lưu trên hệ thống backend của admin (PostgreSQL) phục vụ demo và kiểm tra idempotency.

Quyền của người dùng

Admin Page có thể thu hồi quyền truy cập bất cứ lúc nào trong cài đặt Facebook hoặc gỡ ứng dụng khỏi Page.
