Hướng dẫn xóa dữ liệu người dùng

Ứng dụng pageapi lưu sự kiện bình luận/tin nhắn Page phục vụ xử lý phản hồi tự động.

Cách yêu cầu xóa dữ liệu

  1. Gỡ ứng dụng khỏi Page Facebook trong cài đặt Page → Ứng dụng đã kết nối.
  2. Hoặc gửi email tới admin kèm Page ID và User ID (PSID) cần xóa.

Thời gian xử lý

Yêu cầu được xử lý trong vòng 30 ngày. Dữ liệu liên quan trong cơ sở dữ liệu backend sẽ bị xóa hoặc ẩn danh hóa.
